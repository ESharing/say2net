Welcome to the Kaazing Gateway Demos

Kaazing Gateway 8.12 (Battlestar) Release

Kaazing Gateway is a WebSocket server, featuring innovative solutions and implementations of the HTML 5 Web sockets and Server-sent events (SSE) specification. With Kaazing Gateway you can easily develop and deploy Web applications using protocol-specific client libraries that implement server-side protocols, such as JMS, JMX, SMTP, IMAP, and Jabber. Kaazing's JavaScript libraries support all major browsers (Firefox 1.5+, Internet Explorer 5.5+, Safari 3.0+, Opera 9.5+ and Google Chrome 0.2+).

To be able to run these demos you need to have the Kaazing Gateway demo bundle and its Apache ActiveMQ instance up-and-running - Refer to the Getting Started Guide for more information on how to setup the demo bundle.
  • JavaScript Stomp

    This demo illustrates the use of Kaazing's API specific JavaScript implementation of the Stomp specification and how users can subscribe and send messages to certain destinations

  • Stock Trading

    This demo illustrates how the Kaazing Gateway can be used to stream data to Web clients using Apache ActiveMQ as a message broker and Stomp as the wire protocol.

  • XMPP Chat

    This demo illustrates how the Kaazing Gateway uses the XMPP protocol to allow web applications to talk to a variety of chat servers.

  • Server-Sent Events

    This simple demo provides an example of the HTML 5 Server-Sent Events feature, provided today by the Kaazing Gateway.

  • PostMessage

    This demo shows how the Kaazing Gateway offers the HTML 5 postMessage() feature for your current applications.
